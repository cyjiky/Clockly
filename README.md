## Run

To run the application, follow these steps:

**Copy the repository:**
```bash
git clone https://github.com/cyjiky/LessSocial-Gym.git
```

**Move to repository directory:**
```bash
cd ...
```

**Run `compose.yaml`**
```bash
docker compose up
```
