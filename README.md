## Run

To run the application, follow these steps:

**Copy the repository:**
```bash
git clone https://github.com/cyjiky/Clockly.git
```

**Move to repository directory:**
```bash
cd Clock
```

**Run `compose.yaml`**
```bash
docker compose up
```
